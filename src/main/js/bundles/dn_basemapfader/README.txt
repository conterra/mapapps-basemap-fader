Dieses Bundle kann NICHT zusammen mit dem 

1. basemaptoggler oder basemapgallery Bundle in einer App verwendet werden.
Die basemapgallery stellt bei jedem MapModel structure change event sicher, dass nur ein Baselayer enabled ist.
Das basemapfader Bundle aktiviert jedoch zwei Basemaplayer gleichzeitig.

2. followme Bundle verwendet werden: Die Opazität der Basiskarten wird per CSS gesteuert,
welches nicht übertragen wird.
